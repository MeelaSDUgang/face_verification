# 🔍 Сервис верификации лиц

Самостоятельно размещаемый микросервис для верификации лиц на базе **FastAPI + DeepFace**.

Сервис развернут в облачной инфраструктуре **RunPod** и предоставляет REST API для регистрации и проверки пользователей по фотографии.

Без сторонних сервисов распознавания лиц и без передачи данных третьим лицам.

---

# Возможности

* Регистрация эталонного лица пользователя (сохраняется только вектор эмбеддинга, фотография не хранится)
* Проверка новой фотографии на совпадение с зарегистрированным лицом
* Настраиваемые модели (`Facenet512`, `ArcFace`, `VGG-Face` и другие)
* Облачное размещение в RunPod

---

# API

Все эндпоинты поддерживают заголовок:

```http
X-Api-Key: <ваш_ключ>
```

если в конфигурации задан параметр `API_KEY`.

---

## GET /health

Возвращает статус сервиса и количество зарегистрированных пользователей.

### Ответ

```json
{
  "status": "ok",
  "model": "Facenet512",
  "detector": "retinaface",
  "registered_users": 3
}
```

---

## POST /register/{user_id}

Регистрирует эталонное лицо пользователя.

### Тело запроса

```multipart/form-data
file=<JPEG или PNG изображение>
```

### Пример

```bash
curl -X POST <API_URL>/register/alice \
  -F "file=@/path/to/alice_photo.jpg"
```

### Ответ

```json
{
  "success": true,
  "user_id": "alice",
  "message": "Лицо успешно зарегистрировано."
}
```

---

## POST /verify/{user_id}

Проверяет, соответствует ли переданное изображение ранее зарегистрированному пользователю.

### Тело запроса

```multipart/form-data
file=<JPEG или PNG изображение>
```

### Пример

```bash
curl -X POST <API_URL>/verify/alice \
  -F "file=@/path/to/live_photo.jpg"
```

### Ответ

```json
{
  "success": true,
  "verified": true,
  "user_id": "alice",
  "confidence": 0.923,
  "distance": 0.187,
  "threshold": 0.40,
  "elapsed_ms": 312.4
}
```

| Поле         | Описание                                 |
| ------------ | ---------------------------------------- |
| `verified`   | `true`, если лица совпадают              |
| `confidence` | Уровень уверенности от 0 до 1            |
| `distance`   | Косинусное расстояние между эмбеддингами |
| `threshold`  | Пороговое значение принятия решения      |
| `elapsed_ms` | Время обработки запроса в миллисекундах  |

---

## DELETE /users/{user_id}

Удаляет сохранённый эмбеддинг пользователя.

### Пример

```bash
curl -X DELETE <API_URL>/users/alice
```

## GET /users

Возвращает список зарегистрированных пользователей.

# Интеграция (Python)

Скопируйте файл `face_verify_client.py` в свой проект:

```python
from face_verify_client import FaceVerifyClient

client = FaceVerifyClient(
    "<API_URL>",
    api_key="your-key"
)

# Регистрация пользователя
client.register(
    "user_123",
    "/uploads/onboarding_photo.jpg"
)

# Проверка личности
result = client.verify(
    "user_123",
    "/tmp/photo.jpg"
)

if result.verified:
    print(
        f"✓ Личность подтверждена "
        f"({result.confidence:.0%} уверенности)"
    )
else:
    print(
        f"✗ Лицо не совпадает "
        f"(distance={result.distance:.3f})"
    )
```

# Конфигурация

Все параметры задаются через переменные окружения.

| Переменная             | По умолчанию         | Описание                                                             |
| ---------------------- | -------------------- | -------------------------------------------------------------------- |
| `MODEL_NAME`           | `Facenet512`         | Используемая модель (`VGG-Face`, `Facenet`, `Facenet512`, `ArcFace`) |
| `DETECTOR_BACKEND`     | `retinaface`         | Детектор лица (`retinaface`, `mtcnn`, `opencv`, `ssd`)               |
| `SIMILARITY_THRESHOLD` | `0.40`               | Порог косинусного расстояния                                         |
| `DB_PATH`              | `data/embeddings.db` | Путь к SQLite-базе данных                                            |
| `API_KEY`              | отсутствует          | Включает обязательную авторизацию по API-ключу                       |

---

# Выбор модели

| Модель       | Точность      | Скорость     | Рекомендации              |
| ------------ | ------------- | ------------ | ------------------------- |
| `VGG-Face`   | Хорошая       | Высокая      | Быстрые проверки          |
| `Facenet`    | Очень хорошая | Средняя      | Универсальный вариант     |
| `Facenet512` | Отличная      | Средняя      | **Рекомендуемый вариант** |
| `ArcFace`    | Отличная      | Ниже средней | Максимальная точность     |

# Настройка порога

| Значение | Поведение                    |
| -------- | ---------------------------- |
| `0.30`   | Очень строгая проверка       |
| `0.40`   | **Сбалансированный вариант** |
| `0.50`   | Более мягкая проверка        |
